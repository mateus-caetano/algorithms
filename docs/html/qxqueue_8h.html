◆ qx_queue_back()

int qx_queue_back ( QxQueue  queue)

Determina o valor do elemento no fundo (fim) de uma fila.

Ou seja, determina qual foi o último elemento enfileirado por uma chamada a qx_queue_enqueue.

Parâmetros
queueFila a consultar.
Retorna
Valor do elemento no fim de queue, caso exista; QX_LAMBDA_ELEMENT caso queue esteja vazia.

◆ qx_queue_find()

size_t qx_queue_find ( QxQueue  queue,
int  element 
)

Determina numa fila quantos elementos existem à frente de um valor informado.

Ou seja, quantos elementos temos que desenfileirar de forma a obtermos um elemento com valor igual ao informado na frente da fila.

Parâmetros
queueFila a consultar.
elementValor a procurar.
Retorna
Quantidade de elementos à frente do procurado; ou QX_INFINITY_SIZE caso valor não exista em queue.

◆ qx_queue_font()

int qx_queue_font ( QxQueue  queue)

Determina o valor do elemento na frente (início) de uma fila.

Ou seja, determina qual seria o próximo elemento a ser removido por uma chamada a qx_queue_dequeue.

Parâmetros
queueFila a consultar.
Retorna
Valor do elemento no início de queue, caso exista; QX_LAMBDA_ELEMENT caso queue esteja vazia.
