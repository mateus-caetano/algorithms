◆ QxStack

typedef struct __qxd_stack* QxStack

Representa uma Pilha. O tipo de pilha (estática ou dinâmica) é definido pela implementação.

Funções

◆ qx_stack_capacity()

size_t qx_stack_capacity ( QxStack  stack)

Determina a capacidade máxima atual da pilha.

A diferença entre a capacidade e o tamanho determina o número de elementos que ainda podemos inserir sem causar uma realocação de memória na pilha.

Para implementações que não limitam a capacidade da pilha de forma rígida, deve sempre retornar QX_INFINITY_SIZE.

Parâmetros
stackPilha a consultar.
Retorna
Número de espaços existentes para elementos na pilha; ou QX_INFINITY_SIZE.

◆ qx_stack_full()

bool qx_stack_full ( QxStack  stack)

Determina se o vetor está com sua capacidade completamente ocupada por elementos.

Ou seja, caso a resposta seja true, não há espaço livre para novos elementos, de forma que uma nova inserção causaria uma realocação de memória no vetor.

Para implementações que não limitam a capacidade da pilha de forma rígida, deve sempre retornar false.

Parâmetros
stackPilha a consultar.
Retorna
true caso stack informada não possui espaço livre; false caso contrário.
