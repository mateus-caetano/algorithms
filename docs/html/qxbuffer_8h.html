◆ qx_buffer_full()

bool qx_buffer_full ( QxBuffer  buffer)

Determina se o buffer está com sua capacidade completamente ocupada por elementos.

Ou seja, caso a resposta seja true, não há espaço livre para novos elementos, de forma que uma nova inserção falharia.

Parâmetros
bufferBuffer a consultar.
Retorna
true caso buffer informado não possua espaço livre; false caso contrário.

◆ qx_buffer_move()

size_t qx_buffer_move ( QxBuffer  buffer,
QxBuffer  to 
)

Move os elementos de um buffer para outro.

Os elementos são movidos na ordem em que entraram no buffer. A movimentação acaba quando:

  1. o buffer original esvaziar; ou
  2. o buffer destino encher.

Os elementos que não foram movidos permanecem no buffer de origem.

Parâmetros
bufferBuffer de onde os elementos sairão.
toBuffer para onde os elementos serão movidos.
Retorna
Quantidade de elementos movidos.
