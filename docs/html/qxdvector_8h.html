size_t qx_vector_capacity ( QxVector  vector)

Determina a capacidade máxima atual do vetor.

A diferença entre a capacidade e o tamanho determina o número de elementos que ainda podemos inserir sem causar uma realocação de memória no vetor.

Parâmetros
vectorVetor a consultar.
Retorna
Número de espaços existentes para elementos no vetor.

◆ qx_vector_full()

bool qx_vector_full ( QxVector  vector)

Determina se o vetor está com sua capacidade completamente ocupada por elementos.

Ou seja, caso a resposta seja true, não há espaço livre para novos elementos, de forma que uma nova inserção causaria uma realocação de memória no vetor.

Parâmetros
vectorVetor a consultar.
Retorna
true caso vector informado não possua espaço livre; false caso contrário.
